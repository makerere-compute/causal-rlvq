Datasets : 
==========

1. The synthetic is generated from the 'gen_simple_network' this also automatically calls the z_transformation code to transform the data.
2. The lucas dataset is in data/lucas - the matlab format has all the three datasets.
3. To run the code, call 'main'. You can also set the parameters from 'main'
4. getresults.m combines all the results into the table in the paper.
